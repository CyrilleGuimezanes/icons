# Icons

Fun Unity game based on Google Icons, fully generated by AI.

## Project Setup

This is a **Unity 6** (6000.0.39f1) **2D game** project configured for **vertical (portrait) mode**.

### Key Configuration

- **Unity Version**: 6000.0.39f1 (Unity 6)
- **Render Mode**: 2D
- **Screen Orientation**: Portrait (Vertical)
- **Default Resolution**: 1080x1920 (9:16 aspect ratio)
- **Physics**: 2D Physics enabled

### Getting Started

1. Open the project in Unity 6 or later
2. The main scene is located at `Assets/Scenes/MainScene.unity`
3. The project is pre-configured for portrait/vertical mode on mobile devices

### Project Structure

```
├── Assets/
│   └── Scenes/
│       └── MainScene.unity    # Main game scene
├── Packages/
│   └── manifest.json          # Package dependencies
├── ProjectSettings/           # Unity project settings
└── README.md
```

### Features

- 2D game setup with orthographic camera
- Portrait/vertical orientation for mobile gameplay
- Input system configured for touch and keyboard controls
- Physics2D settings ready for 2D game development
